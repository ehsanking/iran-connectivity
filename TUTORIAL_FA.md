# ابزار تحلیل اتصال اینترنت ایران - آموزش فارسی

## 🎯 کاربرد این ابزار

این ابزار برای فعالان آزادی اینترنت طراحی شده است تا دیتاسنترها، CDNها و ارائه‌دهندگان DNS را شناسایی کنند که در زمان محدودیت‌های اینترنت بین‌المللی همچنان می‌توانند به سرورهای ایرانی متصل شوند. این ابزار تحلیل جامع و پیشنهادهای تونل‌زنی برای حفظ ارتباط ارائه می‌دهد.

## 🚀 راهنمای سریع شروع

### گام ۱: نصب ابزار
```bash
git clone https://github.com/ehsanking/iran-connectivity.git
cd iran-connectivity
npm install
```

### گام ۲: تحلیل پایه
```bash
# تست اتصال به یک IP ایرانی
node cli.js analyze 185.185.123.45
```

### گام ۳: درک نتایج
ابزار به شما نشان می‌دهد:
- کدام ارائه‌دهندگان می‌توانند به IP ایرانی برسند
- امتیاز کیفیت اتصال (۰-۱۰۰)
- پروتکل‌های تونل‌زنی پیشنهادی
- در دسترس بودن پورت‌ها (۸۰، ۴۴۳، ۲۲، ۵۳)

## 📊 دستورات در دسترس

### ۱. تحلیل اتصال
```bash
# تحلیل پایه
node cli.js analyze <IP_ایرانی>

# تحلیل دقیق با زمان انتظار سفارشی
node cli.js analyze 185.185.123.45 --timeout 10 --detailed

# خروجی گرفتن نتایج به CSV
node cli.js analyze 185.185.123.45 --export csv --output results.csv
```

### ۲. لیست ارائه‌دهندگان
```bash
# نمایش تمام ارائه‌دهندگان در دسترس
node cli.js providers

# فیلتر بر اساس منطقه
node cli.js providers --region "middle-east"
```

### ۳. دریافت پیشنهادهای تونل‌زنی
```bash
# تولید پیشنهادهای تونل‌زنی
node cli.js recommend connectivity_report.json
```

### ۴. مدیریت پیکربندی
```bash
# نمایش تنظیمات فعلی
node cli.js config show

# تنظیم زمان انتظار سفارشی
node cli.js config set timeout 15

# افزودن رنج‌های IP سفارشی
node cli.js config add-range "192.168.1.0/24" "My Provider"
```

## 🔒 ویژگی‌های امنیتی

- **تست محلی**: تمام تست‌ها به‌صورت محلی روی سرور شما اجرا می‌شود
- **لاگ‌های رمزنگاری شده**: داده‌های حساس رمزنگاری می‌شوند
- **محدودیت نرخ**: جلوگیری از شناسایی
- **عملیات مخفی**: ردپای شبکه‌ای کم

## 🛠️ پروتکل‌های تونل‌زنی پشتیبانی شده

### پروتکل‌های پیشنهادی
۱. **WireGuard** - سریع و امن
۲. **OpenVPN** - قابل اعتماد و پشتیبانی گسترده
۳. **SSH** - ساده و مؤثر
۴. **Shadowsocks** - مناسب برای دور زدن محدودیت‌ها
۵. **V2Ray** - مبهم‌سازی پیشرفته

### روش‌های انتقال
- **TCP-over-TCP**: قابل اعتماد اما کندتر
- **UDP-over-TCP**: بهتر برای استریمینگ
- **WebSocket over TLS**: سخت برای تشخیص
